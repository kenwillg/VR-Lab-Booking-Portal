PWA Icon Generator

Gunakan tool online untuk membuat icon PWA:

  • RealFaviconGenerator
  • PWA Builder Image Generator
  • Favicon.io

Atau buat icon sederhana dengan teks "VR Lab" menggunakan design tool seperti:

  • Canva
  • Figma
  • Photoshop

Requirements:

  • 192x192 pixels (pwa-192x192.png)
  • 512x512 pixels (pwa-512x512.png)
  • Format: PNG
  • Background: Transparent atau solid color
